# npm

## 1. 初始化一个新项目

```
npm init
```

## 2. 安装依赖包

```
npm i [package]
npm i [package]@[version]
```

## 3. 全局安装依赖

```
npm i -g [package]
```

## 4. 将依赖项添加到不同依赖项类别中

```
npm i [package]
npm i [package] -D
```

## 5. 移除依赖包

```
npm rm [package]
```

## 6. 安装项目的全部依赖

```
npm i
```

## 7. 登录 registry

```
npm login
```

## 8. 退出 registry

```
npm logout
```

## 9. 发布包

```
npm publish
```

## 10. 使用淘宝源

```
npm config set registry https://registry.npmmirror.com
npm get registry
```

## 11. 改回源

```
npm config set registry https://registry.npmjs.org
```


## 12. 查看全局包

```
npm list -g
```

## 13. 更新包

```
npm upgrade [package]
```
